# **App Name**: Geometra

## Core Features:

- Asistente Matemático IA: Un asistente de IA que puede ayudar a los usuarios con problemas de matemáticas y preguntas relacionadas con Geogebra. La IA actúa como una herramienta para razonar sobre qué información específica del dominio compartir con el usuario, según sus necesidades.
- Applet Interactivo de Geogebra: Un applet de Geogebra de pantalla completa para la exploración matemática interactiva.
- Página de Inicio: Página de inicio que proporciona información clara sobre lo que ofrece el sitio web.
- Navegación: Navegación fácil para ayudar a los usuarios a navegar entre la sección de ayuda matemática y el applet interactivo.

## Style Guidelines:

- Color primario: HSL 220, 50%, 60% (Hex: #4D79B3), un azul calmante e intelectual, que evoca conocimiento y concentración.
- Color de fondo: HSL 220, 20%, 95% (Hex: #F0F2F5), un azul muy claro y desaturado que proporciona un fondo limpio y discreto.
- Color de acento: HSL 190, 60%, 50% (Hex: #26A69A), un verde azulado vibrante para resaltar elementos interactivos y llamadas a la acción, atrayendo la atención sin abrumar.
- Combinación de fuentes: 'Space Grotesk' (sans-serif) para encabezados, combinada con 'Inter' (sans-serif) para el texto del cuerpo, que ofrece una sensación moderna y legible.
- Utilice iconos simples y claros para representar diferentes conceptos matemáticos y funciones de Geogebra.
- Diseño limpio y organizado con un amplio espacio en blanco para evitar el desorden visual y mejorar la legibilidad.
- Animaciones sutiles para proporcionar retroalimentación y guiar al usuario a través de las interacciones.
