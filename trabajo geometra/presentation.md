# La Problemática
## El Aprendizaje Solitario

- **El Problema**: Aprender matemáticas solo en casa genera frustración.
- **La Brecha**: Herramientas como GeoGebra grafican, pero no explican el "por qué" de un error.
- **La Solución**: Un **Tutor IA** que acompaña y retroalimenta en tiempo real.

# Matemáticas en el Código
## Vectores

- **Vectores en el Desarrollo**:
    - No solo son materia de estudio, ¡son la base de la app!
    - **Posicionamiento**: Cada botón y ventana usa coordenadas $(x, y)$.
    - **Movimiento**: Las animaciones son traslaciones vectoriales $\vec{v} = (dx, dy)$.

# Matemáticas en el Código
## Homotecia y Funciones

- **Homotecia = Diseño Responsivo**:
    - Para adaptar la web a celulares, aplicamos **Homotecia**.
    - Multiplicamos las dimensiones por un factor de escala $k$ ($k < 1$ para móviles).
- **Funciones**:
    - El código es pura función: $Entrada \rightarrow Proceso \rightarrow Salida$.
    - Ejemplo: `calcularArea(radio)` es una función $f(x)$.
