## Diapositiva 5: Matemáticas en el Código (Vectores)
**Texto**:
*   **Vectores en el Desarrollo**:
    *   No solo son materia de estudio, ¡son la base de la app!
    *   **Posicionamiento**: Cada botón y ventana usa coordenadas $(x, y)$.
    *   **Movimiento**: Las animaciones son traslaciones vectoriales $\vec{v} = (dx, dy)$.
**Prompt de Imagen (Nano Banana)**: "Wireframe of a website interface with visible grid lines and vector coordinates (x,y) marking the position of elements, blueprint style, technical drawing, cyan and white lines."

## Diapositiva 6: Matemáticas en el Código (Homotecia y Funciones)
**Texto**:
*   **Homotecia = Diseño Responsivo**:
    *   Para adaptar la web a celulares, aplicamos **Homotecia**.
    *   Multiplicamos las dimensiones por un factor de escala $k$ ($k < 1$ para móviles).
*   **Funciones**:
    *   El código es pura función: $Entrada \rightarrow Proceso \rightarrow Salida$.
    *   Ejemplo: `calcularArea(radio)` es una función $f(x)$.
**Prompt de Imagen (Nano Banana)**: "Visual analogy of Homothety, a large computer screen and a small phone screen showing the same geometric interface perfectly scaled, connected by projection lines, isometric 3d view."
