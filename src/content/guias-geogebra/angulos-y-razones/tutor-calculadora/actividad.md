### Configuración de la Calculadora

Revisa el modo de tu calculadora. Las opciones son:
- **Grados:** `DEG`
- **Radianes:** `RAD`
- **Gradianes:** `GRA`

---
### PARTE 1: Modo Grados (DEG)
*Configura tu calculadora en modo `DEG`.*

#### **ACTIVIDAD 10: Calcular Seno (en grados)**
Para calcular `sin(30°)`, presiona:
`SIN` -> `3` `0` -> `)` (si es necesario) -> `=`.
El resultado debe ser `0.5`.

#### **ACTIVIDAD 11: Calcular Coseno (en grados)**
Para calcular `cos(30°)`, presiona:
`COS` -> `3` `0` -> `)` (si es necesario) -> `=`.
El resultado debe ser aprox. `0.8660`.

#### **ACTIVIDAD 12: Calcular Tangente (en grados)**
Para calcular `tan(30°)`, presiona:
`TAN` -> `3` `0` -> `)` (si es necesario) -> `=`.
El resultado debe ser aprox. `0.5774`.

#### **ACTIVIDAD 13: Calcular Ángulo a partir del Seno (en grados)**
Para calcular el ángulo cuyo seno es `0.5`, presiona:
`SHIFT` -> `SIN` (verás `sin⁻¹`) -> `0` `.` `5` -> `)` -> `=`.
El resultado debe ser `30`.

#### **ACTIVIDAD 14: Calcular Ángulo a partir del Coseno (en grados)**
Para calcular el ángulo cuyo coseno es `0.5`, presiona:
`SHIFT` -> `COS` (verás `cos⁻¹`) -> `0` `.` `5` -> `)` -> `=`.
El resultado debe ser `60`.

#### **ACTIVIDAD 15: Calcular Ángulo a partir de la Tangente (en grados)**
Para calcular el ángulo cuya tangente es `0.5`, presiona:
`SHIFT` -> `TAN` (verás `tan⁻¹`) -> `0` `.` `5` -> `)` -> `=`.
El resultado debe ser aprox. `26.57`.

---
### PARTE 2: Modo Radianes (RAD)
*Configura tu calculadora en modo `RAD`.*

#### **ACTIVIDAD 16 a 18: Calcular Seno, Coseno y Tangente (en radianes)**
Para calcular las razones de `30 rad`:
- `sin(30)` -> `SIN` `3` `0` `=` -> Resultado: `-0.9880`
- `cos(30)` -> `COS` `3` `0` `=` -> Resultado: `0.1543`
- `tan(30)` -> `TAN` `3` `0` `=` -> Resultado: `-6.4053`

#### **ACTIVIDAD 19 a 21: Calcular Ángulo a partir de la Razón (en radianes)**
Para calcular el ángulo en radianes cuyo valor de seno/coseno/tangente es `0.5`:
- `SHIFT` `SIN` `0.5` `=` -> Resultado: `0.52`
- `SHIFT` `COS` `0.5` `=` -> Resultado: `1.05`
- `SHIFT` `TAN` `0.5` `=` -> Resultado: `0.46`

---
### PARTE 3: Modo Gradianes (GRA)
*Configura tu calculadora en modo `GRA`.*

#### **ACTIVIDAD 22 a 24: Calcular Seno, Coseno y Tangente (en gradianes)**
Para calcular las razones de `50g`:
- `sin(50)` -> `SIN` `5` `0` `=` -> Resultado: `0.7071`
- `cos(50)` -> `COS` `5` `0` `=` -> Resultado: `0.7071`
- `tan(50)` -> `TAN` `5` `0` `=` -> Resultado: `1`

#### **ACTIVIDAD 25 a 27: Calcular Ángulo a partir de la Razón (en gradianes)**
Para calcular el ángulo en gradianes cuyo valor de seno/coseno/tangente es `0.5`:
- `SHIFT` `SIN` `0.5` `=` -> Resultado: `33.33`
- `SHIFT` `COS` `0.5` `=` -> Resultado: `66.67`
- `SHIFT` `TAN` `0.5` `=` -> Resultado: `29.52`
