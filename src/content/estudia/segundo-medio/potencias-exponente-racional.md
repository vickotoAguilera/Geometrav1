# Potencias de Exponente Racional

Una potencia con exponente racional es otra forma de representar una raíz. Esta conexión es fundamental para simplificar operaciones algebraicas complejas.

## Definición

Si $a$ es un número real positivo y $\frac{m}{n}$ es un número racional (donde $n > 0$), entonces:

$$a^{\frac{m}{n}} = \sqrt[n]{a^m} = (\sqrt[n]{a})^m$$

Donde:
- $a$ es la base
- $m$ es el exponente de la potencia
- $n$ es el índice de la raíz

**Ejemplos:**
- $8^{\frac{1}{3}} = \sqrt[3]{8} = 2$
- $4^{\frac{3}{2}} = \sqrt{4^3} = \sqrt{64} = 8$
- $x^{\frac{2}{5}} = \sqrt[5]{x^2}$

## Propiedades

Las propiedades de las potencias se mantienen para exponentes racionales:

1. **Multiplicación de igual base:**
   $$a^p \cdot a^q = a^{p+q}$$

2. **División de igual base:**
   $$\frac{a^p}{a^q} = a^{p-q}$$

3. **Potencia de una potencia:**
   $$(a^p)^q = a^{p \cdot q}$$

4. **Potencia de un producto:**
   $$(a \cdot b)^p = a^p \cdot b^p$$

5. **Potencia de un cociente:**
   $$(\frac{a}{b})^p = \frac{a^p}{b^p}$$

6. **Exponente negativo:**
   $$a^{-p} = \frac{1}{a^p}$$

## Simplificación de Expresiones

Usar exponentes racionales facilita la simplificación de expresiones con raíces anidadas o multiplicaciones de raíces con distinto índice.

**Ejemplo:**
Simplificar $\sqrt{x} \cdot \sqrt[3]{x}$

1. Convertir a potencias: $x^{\frac{1}{2}} \cdot x^{\frac{1}{3}}$
2. Sumar exponentes: $\frac{1}{2} + \frac{1}{3} = \frac{3+2}{6} = \frac{5}{6}$
3. Resultado: $x^{\frac{5}{6}} = \sqrt[6]{x^5}$

## Ejercicios

1. Calcula el valor de $27^{\frac{2}{3}}$
2. Expresa como raíz: $5^{\frac{3}{4}}$
3. Simplifica: $\frac{\sqrt[3]{x^2}}{\sqrt{x}}$
