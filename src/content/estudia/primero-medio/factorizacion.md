## ¿Qué es la Factorización?

Factorizar una expresión algebraica consiste en escribirla como una multiplicación de expresiones más simples (sus factores). Es el proceso inverso de la expansión.

### Casos Comunes de Factorización

1.  **Factor Común Monomio:** Se extrae el máximo factor común de todos los términos.
    - **Ejemplo:** `3x² + 6x = 3x(x + 2)`

2.  **Diferencia de Cuadrados:** Se aplica a binomios que son una resta de dos cuadrados perfectos.
    - **Fórmula:** `a² - b² = (a + b)(a - b)`
    - **Ejemplo:** `x² - 9 = (x + 3)(x - 3)`

3.  **Trinomio Cuadrado Perfecto:** Proviene de un binomio al cuadrado.
    - **Fórmula:** `a² + 2ab + b² = (a + b)²`
    - **Ejemplo:** `x² + 6x + 9 = (x + 3)²`

## Productos Notables

Los productos notables son multiplicaciones algebraicas cuyo resultado se puede escribir directamente, sin necesidad de realizar todo el proceso.

1.  **Cuadrado de un Binomio:**
    - `(a + b)² = a² + 2ab + b²`
    - `(a - b)² = a² - 2ab + b²`

2.  **Suma por su Diferencia:**
    - `(a + b)(a - b) = a² - b²`

3.  **Producto de Binomios con Término Común:**
    - `(x + a)(x + b) = x² + (a + b)x + ab`

## Ejemplo Práctico: Factorizar un Trinomio

**Pregunta:** Factoriza el trinomio `x² + 5x + 6`.

1.  **Analizamos el trinomio:** No es un trinomio cuadrado perfecto. Buscamos dos números que al multiplicarse den `6` (el término `c`) y al sumarse den `5` (el término `b`).

2.  **Buscamos los números:**
    - Pares de números que multiplican 6: (1, 6), (2, 3), (-1, -6), (-2, -3).
    - El par que suma 5 es (2, 3).

3.  **Escribimos la factorización:**
    Usamos el formato del producto de binomios con término común.
    `x² + 5x + 6 = (x + 2)(x + 3)`

**Respuesta:** La factorización es `(x + 2)(x + 3)`.
