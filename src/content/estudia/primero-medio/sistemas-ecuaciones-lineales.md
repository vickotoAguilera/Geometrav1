# Sistemas de Ecuaciones Lineales

Un sistema de ecuaciones lineales es un conjunto de dos o más ecuaciones con las mismas incógnitas. En Primero Medio trabajamos principalmente con sistemas de 2 ecuaciones con 2 incógnitas (2x2).

## Forma General

$$\begin{cases}
ax + by = c \\
dx + ey = f
\end{cases}$$

Donde $x$ e $y$ son las incógnitas que debemos encontrar.

## Métodos de Resolución

### 1. Método de Sustitución

1. Despejar una variable de una ecuación
2. Sustituir en la otra ecuación
3. Resolver para encontrar una variable
4. Sustituir el valor encontrado para hallar la otra variable

**Ejemplo:**
$$\begin{cases}
x + y = 5 \\
2x - y = 1
\end{cases}$$

De la primera ecuación: $y = 5 - x$

Sustituyendo en la segunda: $2x - (5 - x) = 1$
$$2x - 5 + x = 1$$
$$3x = 6$$
$$x = 2$$

Entonces: $y = 5 - 2 = 3$

**Solución:** $(x, y) = (2, 3)$

### 2. Método de Igualación

1. Despejar la misma variable en ambas ecuaciones
2. Igualar las expresiones
3. Resolver para encontrar una variable
4. Sustituir para encontrar la otra

### 3. Método de Reducción (Suma y Resta)

1. Multiplicar las ecuaciones por números convenientes
2. Sumar o restar las ecuaciones para eliminar una variable
3. Resolver la ecuación resultante
4. Sustituir para encontrar la otra variable

**Ejemplo:**
$$\begin{cases}
3x + 2y = 12 \\
x - 2y = 4
\end{cases}$$

Sumando ambas ecuaciones:
$$4x = 16$$
$$x = 4$$

Sustituyendo: $4 - 2y = 4$, entonces $y = 0$

**Solución:** $(x, y) = (4, 0)$

## Tipos de Soluciones

- **Solución única:** Las rectas se intersectan en un punto
- **Infinitas soluciones:** Las rectas son coincidentes
- **Sin solución:** Las rectas son paralelas

## Aplicaciones

Los sistemas de ecuaciones se usan para resolver problemas como:
- Mezclas y combinaciones
- Problemas de edades
- Problemas de velocidad y distancia
- Problemas económicos

## Ejercicios

1. Resuelve por sustitución:
   $$\begin{cases}
   y = 2x + 1 \\
   3x + y = 11
   \end{cases}$$

2. Resuelve por reducción:
   $$\begin{cases}
   2x + 3y = 7 \\
   x - 3y = 2
   \end{cases}$$
