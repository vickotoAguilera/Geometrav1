# Funciones Trigonométricas

Las funciones trigonométricas seno y coseno son fundamentales para modelar fenómenos periódicos y oscilatorios.

## Función Seno

$$f(x) = \sin(x)$$

### Características
- **Dominio:** $\mathbb{R}$ (todos los números reales)
- **Recorrido:** $[-1, 1]$
- **Período:** $2\pi$ (se repite cada $2\pi$ radianes)
- **Amplitud:** 1
- **Puntos notables:**
  - $\sin(0) = 0$
  - $\sin(\pi/2) = 1$
  - $\sin(\pi) = 0$
  - $\sin(3\pi/2) = -1$

### Gráfica
La función seno produce una onda que oscila entre -1 y 1.

## Función Coseno

$$f(x) = \cos(x)$$

### Características
- **Dominio:** $\mathbb{R}$
- **Recorrido:** $[-1, 1]$
- **Período:** $2\pi$
- **Amplitud:** 1
- **Puntos notables:**
  - $\cos(0) = 1$
  - $\cos(\pi/2) = 0$
  - $\cos(\pi) = -1$
  - $\cos(3\pi/2) = 0$

### Relación con Seno
$$\cos(x) = \sin(x + \pi/2)$$

El coseno es una traslación horizontal del seno.

## Transformaciones

### Amplitud
$$f(x) = A\sin(x)$$ o $$f(x) = A\cos(x)$$

La amplitud es $|A|$ y determina la altura máxima de la onda.

### Período
$$f(x) = \sin(Bx)$$ o $$f(x) = \cos(Bx)$$

El período es $\frac{2\pi}{|B|}$

### Desplazamiento Vertical
$$f(x) = \sin(x) + D$$ o $$f(x) = \cos(x) + D$$

Desplaza la gráfica $D$ unidades verticalmente.

### Desplazamiento Horizontal (Fase)
$$f(x) = \sin(x - C)$$ o $$f(x) = \cos(x - C)$$

Desplaza la gráfica $C$ unidades horizontalmente.

## Forma General

$$f(x) = A\sin(B(x - C)) + D$$
$$f(x) = A\cos(B(x - C)) + D$$

Donde:
- $A$ = Amplitud
- $B$ = Frecuencia (período = $2\pi/B$)
- $C$ = Desplazamiento horizontal
- $D$ = Desplazamiento vertical

## Aplicaciones

Las funciones trigonométricas modelan:
- **Ondas sonoras:** Vibraciones y frecuencias
- **Movimiento armónico simple:** Péndulos, resortes
- **Mareas:** Variación del nivel del mar
- **Ciclos biológicos:** Ritmos circadianos
- **Corriente alterna:** Electricidad
- **Ondas electromagnéticas:** Luz, radio

## Identidades Fundamentales

$$\sin^2(x) + \cos^2(x) = 1$$
$$\sin(2x) = 2\sin(x)\cos(x)$$
$$\cos(2x) = \cos^2(x) - \sin^2(x)$$
