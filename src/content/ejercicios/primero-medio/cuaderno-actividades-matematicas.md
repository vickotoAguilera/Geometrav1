2.  **Sustituye los valores W=3/4, X=-4/5, Y=2.4, Z=-2.3... y resuelve.**
    - Z = - (2 + 1/3) = -7/3
    - Y = 2.4 = 12/5
    - a. X * W = (-4/5) * (3/4) = -12/20 = -3/5
    - b. Z * Y = (-7/3) * (12/5) = -84/15 = -28/5
    - c. -W * Z = -(3/4) * (-7/3) = 21/12 = 7/4
    - d. Z * X * Y = (-7/3) * (-4/5) * (12/5) = (28/15) * (12/5) = 336/75 = 112/25
    - e. -W * Z * (-Y) = -(3/4) * (-7/3) * (-12/5) = (7/4) * (-12/5) = -84/20 = -21/5
    - f. 2(-X) * W * Z = 2(4/5) * (3/4) * (-7/3) = (8/5) * (3/4) * (-7/3) = (24/20) * (-7/3) = (6/5) * (-7/3) = -42/15 = -14/5
