### DECIMAL INFINITO SEMIPERIÓDICO

*   Se escribe como numerador la diferencia entre el número decimal (sin la coma) y el número formado por la parte entera y el anteperíodo.
*   En el denominador se escriben tantos 9 como cifras tenga el periodo, seguido de tantos 0 como cifras tenga el anteperíodo.

5,3₂₅ = (5325 - 53) / 990 = 5272 / 990 = 2636 / 495

El denominador es 990 porque el período del número 5,3₂₅ tiene 2 cifras y el anteperíodo 1 cifra.

Un número racional es aquel que puede expresarse como una fracción cuyo numerador es entero y su denominador es un entero distinto de cero.

El conjunto de los **números racionales** (Q) está compuesto por todos los números que se pueden escribir como una fracción, en la cual el numerador y el denominador son números enteros. Además, el denominador debe ser distinto de cero.

Q = {a/b / a, b ∈ Z, b ≠ 0}

Por ejemplo ½, -1,8; 5; -¾.

## EJEMPLO 2

En una municipalidad se lanzó una campaña para recolectar celulares en desuso y así poder reciclar sus componentes. En la primera semana se recogieron $\frac{3}{4}$ kg y en la segunda $1\frac{1}{2}$ kg. ¿Cuántos kilogramos se juntaron en las dos semanas?

Para responder a la pregunta, expresa el número mixto como una fracción y resuelve la adición.
$\frac{3}{4} + 1\frac{1}{2} = \frac{3}{4} + \frac{3}{2} = \frac{3 \cdot 2 + 4 \cdot 3}{4 \cdot 2} = \frac{6 + 12}{8} = \frac{18}{8} = \frac{9}{4} = 2\frac{1}{4}$
Se recolectaron $2\frac{1}{4}$ kg.

---
Para transformar un número mixto a fracción: $6\frac{7}{8} = \frac{6 \cdot 8 + 7}{8} = \frac{55}{8}$
---

Para resolver una **adición** o **sustracción de números decimales**, puedes ordenarlos de manera vertical, cuidando que la coma decimal quede alineada.
Para resolver una **adición** o **sustracción de fracciones**:
Adición: $\frac{a}{b} + \frac{c}{d} = \frac{ad + bc}{bd}$
Sustracción: $\frac{a}{b} - \frac{c}{d} = \frac{ad - bc}{bd}$
(b, d ≠ 0)

## EJEMPLO 3

Calcula el resultado de $2.\overline{3} - \frac{4}{5}$.
Transforma el decimal a fracción: $2.\overline{3} = \frac{23-2}{9} = \frac{21}{9}$
Resuelve: $\frac{21}{9} - \frac{4}{5} = \frac{21 \cdot 5 - 9 \cdot 4}{9 \cdot 5} = \frac{105 - 36}{45} = \frac{69}{45} = \frac{23}{15} = 1\frac{8}{15}$

Para sumar y restar números racionales se puede utilizar su representación fraccionaria o decimal. Si hay números decimales infinitos, siempre debes transformarlos a fracción.

## EJEMPLO 2
Calcula el producto entre $X = -\frac{7}{6}$ e $Y = 2.\overline{46}$.
$2.\overline{46} = \frac{222}{90} = \frac{37}{15}$
$X \cdot Y = -\frac{7}{6} \cdot \frac{37}{15} = -\frac{259}{90} = -2.\overline{87}$

---
Para multiplicar **decimales**, se multiplican como naturales y se ajusta la coma.
Para multiplicar **fracciones**, $\frac{a}{b} \cdot \frac{c}{d} = \frac{ac}{bd}$.
---

## EJEMPLO 3
Calcula $\frac{10}{8} : 3.\overline{6}$.
$3.\overline{6} = \frac{33}{9} = \frac{11}{3}$
$\frac{10}{8} : \frac{11}{3} = \frac{10}{8} \cdot \frac{3}{11} = \frac{30}{88} = \frac{15}{44}$

Para dividir fracciones, se multiplica por el inverso multiplicativo del divisor.
$\frac{a}{b} : \frac{c}{d} = \frac{a}{b} \cdot \frac{d}{c}$

### DECIMAL INFINITO SEMIPERIÓDICO

*   Se escribe como numerador la diferencia entre el número decimal (sin la coma) y el número formado por la parte entera y el anteperíodo.
*   En el denominador se escriben tantos 9 como cifras tenga el periodo, seguido de tantos 0 como cifras tenga el anteperíodo.
*   Si es posible, se simplifica.

5,3₂₅ = (5325 - 53) / 990 = 5272 / 990 = 2636 / 495

El denominador es 990 porque el período del número 5,3₂₅ tiene 2 cifras y el anteperíodo 1 cifra.

Un número racional es aquel que puede expresarse como una fracción cuyo numerador es entero y su denominador es un entero distinto de cero.

El conjunto de los **números racionales** (Q) está compuesto por todos los números que se pueden escribir como una fracción, en la cual el numerador y el denominador son números enteros. Además, el denominador debe ser distinto de cero.

Q = {a/b / a, b ∈ Z, b ≠ 0}

Por ejemplo ½, -1,8; 5; -¾.

## EJEMPLO 2

En una municipalidad se lanzó una campaña para recolectar celulares en desuso y así poder reciclar sus componentes. En la primera semana se recogieron $\frac{3}{4}$ kg y en la segunda $1\frac{1}{2}$ kg. ¿Cuántos kilogramos se juntaron en las dos semanas?

Para responder a la pregunta, expresa el número mixto como una fracción y resuelve la adición.
$\frac{3}{4} + 1\frac{1}{2} = \frac{3}{4} + \frac{3}{2} = \frac{3 \cdot 2 + 4 \cdot 3}{4 \cdot 2} = \frac{6 + 12}{8} = \frac{18}{8} = \frac{9}{4} = 2\frac{1}{4}$
Se recolectaron $2\frac{1}{4}$ kg.

---
Para transformar un número mixto a fracción: $6\frac{7}{8} = \frac{6 \cdot 8 + 7}{8} = \frac{55}{8}$
---

Para resolver una **adición** o **sustracción de números decimales**, puedes ordenarlos de manera vertical, cuidando que la coma decimal quede alineada.
Para resolver una **adición** o **sustracción de fracciones**:
Adición: $\frac{a}{b} + \frac{c}{d} = \frac{ad + bc}{bd}$
Sustracción: $\frac{a}{b} - \frac{c}{d} = \frac{ad - bc}{bd}$
(b, d ≠ 0)

Considera que $X = -\frac{7}{6}$, $Y = 2.\overline{46}$. ¿Cuál es el producto entre X e Y?

## EJEMPLO 3

¿Cuál es el resultado de $\frac{10}{8} : 3.\overline{6}$ ?
